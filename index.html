RooM8

RooM8 is a currently in development application that began as part of a 5 day sprint at General Assembly. The application takes in user preferences and matches them with potential roommates. Built with Rails, Ember.js, JQuery, Postgres + More.

DefaceNYC

DefaceNYC is an application built using a Sinatra server along with a Backbone/JQuery front end. It as built as poart of a group project at General Assembly over a 4 day sprint.

ChampionGraph

ChampionGraph is a simple Sinatra based application that graphs comparisons of records of two previous NBA champions at any point over the course of an 82 game season. Data was grabbed from BasketballReference.com and moved to a Postgres database. Graphing was done with the Gruff gem.

Infinite Tic Tac Toe

Infinite Tic Tac Toe is a version of the game Tic Tac toe written using vanilla javascript. It was really fun to implement the math neccesary to calulate winners on a board of any size in my very early days of learning javascript.

Contact List

This is a contact list application created using Backbone.js,JQuery,and JQuery UI on the front end, with a Ruby Sinatra backend. The app utilizes the Google Maps API as well as the Twilio API and the Hipchat API to allow users to send messages to their contacts directly from the app
